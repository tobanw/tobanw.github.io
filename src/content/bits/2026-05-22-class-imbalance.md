I published a new essay: [Stop worrying about class imbalance](/writing/stop-worrying-about-class-imbalance/).

The short version: if you care about calibrated probabilities, log loss already treats the base rate correctly. Resampling can be useful for a deliberate decision objective, but it is not a neutral fix for an imbalanced training set.
